## Reverse Proxy Recommendation

### Pangolin
**Website**: https://fossorial.io/
**Repository**: https://github.com/fosrl/pangolin
**Documentation**: https://docs.fossorial.io/

For exposing your services securely to the internet, we highly recommend **Pangolin** - a self-hosted alternative to Cloudflare Tunnels.

**Why Pangolin?**
- Self-hosted and open source (AGPL-3)
- WireGuard-based tunneling (no port forwarding required)
- Built-in identity and access management
- Automatic SSL certificates via Let's Encrypt
- Beautiful, intuitive web interface
- CrowdSec integration for security

**Support the project**:
- Star the repository
- Contribute features or documentation
- Report bugs and provide feedback
- [Commercial licensing](https://fossorial.io/) for enterprise use

## How to Contribute to This Project

### Reporting Issues
1. Check if the issue already exists
2. Provide detailed reproduction steps
3. Include relevant logs and configuration (sanitize sensitive data)

### Suggesting Improvements
1. Open a discussion or issue
2. Explain the use case and benefits
3. Provide examples if possible

### Submitting Changes
1. Fork the repository
2. Create a feature branch
3. Test your changes thoroughly
4. Submit a pull request with clear description

### Documentation
- Fix typos and improve clarity
- Add examples and use cases
- Translate to other languages
